Projects

Predicting Micro-Loan Defaults on Prosper.com

This started as a personal project attempting to find loans with high rates of return which, while classified as risky by Prosper.com, were in fact likely to result in profit. Unfortunately, the data provided by Prosper.com didn't contain a unique identifier connecting the loan data with the data on the individuals requesting the loan. So, I only used loan characteristics. To make matters more difficult, most of the loan characteristics were highly correlated. In spite of all that, however, using a handful of predictors I was still able to predict profit or loss with 74% accuracy using gradient boosted trees.

Deducing the Origin of Facebook Confessions Board Posts Using Only Sentiment

This is one of my favorite projects. To start there was no objective, only text data acquired from a Facebook API. To find a problem of interest I spent a lot of time working with unsupervised learning (primarily principal components analysis). This led to eventually testing some oddly accurate predictive random forest models which were used reductively to infer the power of sentiments in predicting the country of origin of each post.

Predicting Crime in Austin, TX

This was a school project which predicts crime in Austin, TX for the year 2016 by zip code. What makes this interesting is that there were more characteristics for each zip code than zip codes. This led to a problem with the curse of dimensionality. To overcome the issue, I originally tried tree based models and ensemble methods to automatically perform feature selection. Though, I eventually decided to prune the features with subset selection methods and found a linear model (Lasso) to have the most predictive power.

College Football Scraper & Analysis

This project started with a grandiose idea: what if I could predict the outcome or spread of college football games? To that end I scraped the last decade of college football team, player, and game data and prepared to begin modeling. But, for various reasons, I shelved the project. This will be a priority when I can find the time. As it currently stands, I've left it as a Pandas (Python) tutorial.
