Projects

Predicting Micro-Loan Defaults on Prosper.com

Using only Amount Borrowed, Borrower Rating, Length of the Loan, and Interest Rate, was able to predict whether a loan would default with 74% accuracy with a gradient boosted tree.

Predicting Crime in Austin, TX

Predicted crime in Austin, TX for the year 2016 by zip code. An rmse of 375 was achieved using Lasso. Should historical crime data be added from the years 2014 and 2015 the rmse drops to 154.

Deducing the Origin of Facebook Confessions Board Posts Using Only Sentiment

Built a classifier to categorize the country of origin of posts on Facebook Confessions Board using only sentiment. Random forest and boosted trees resulted in a model accuracy of +90%.

College Football Scraper & Analysis

Used Python to retrieve college football team, player, and game statistics from the web, store data in a SQLite database, and perform analyses using Python’s NumPy, Pandas, and Matplotlib libraries. Since this no longer being maintained, it is now a tutorial on the pandas library.
